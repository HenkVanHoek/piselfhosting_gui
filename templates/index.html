Welcome to the PiSelfhosting Component Management GUI

Use this application to manage the metadata of your PiSelfhosting components. Select a component on the left to edit its details, or add a new component.

Important Notes:

  • Each Component ID must be unique and can only contain lowercase letters, numbers, and hyphens.
  • If a component has a Web Interface, the UI Port, Protocol, Icon, Dashy Section, and Dashy URL Suffix fields are mandatory.
  • **The UI Port must be unique for all components with a Web Interface.**
  • **Only one component can be marked as 'Is Reverse Proxy'.**
  • After editing, save to store changes to the components_metadata.json file.
  • After saving changes, you must run your `piselfhosting_installer.py` Orchestrator to push the updates to your Raspberry Pi and reconfigure the services.
